# Examples

Examples are intentionally documentation-only. They show how to prepare a project before implementation.

## Demo Student Management

Path:

```text
examples/demo-student-management/
```

Use when learning the Standard/Enterprise track for a web admin app with sensitive personal data.

Includes:

- Product Passport
- PRD
- Architecture
- Research report
- UX spec
- Data/API contract
- Epics
- Story packet
- Test matrix
- Readiness review
- Agent brief

## Demo CLI

Path:

```text
examples/demo-cli/
```

Use when learning the Quick/Standard track for a non-UI tool.

## Demo Automation

Path:

```text
examples/demo-automation/
```

Use when learning input/output contracts, retries, logs, and operational evidence.

## How To Read A Demo

1. Start with `README.md`.
2. Read `docs/product/product-passport.yaml`.
3. Read `docs/product/prd.md`.
4. Read `docs/architecture.md`.
5. Read `docs/stories/`.
6. Read `docs/TEST_MATRIX.md`.
7. Read `docs/readiness-review.md`.
