# Frontend SEO

## Metadata
- Admin app pages use concise titles for browser tabs.
- Login page has safe public metadata without student data.
- Student detail pages never expose personal data in title or metadata.

## Canonical URLs
- Public login page has a canonical URL.
- Authenticated application routes are excluded from public indexing.

## Sitemap And Robots
- Authenticated pages are not included in sitemap.
- Robots policy prevents indexing admin routes.
- Public documentation or landing pages must be added deliberately.
