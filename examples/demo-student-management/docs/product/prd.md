# PRD: Student Management System

## Problem

Training centers often manage students, classes, attendance, and tuition across spreadsheets and chat threads.

## MVP Scope

- Admin login.
- Student profile CRUD.
- Class management.
- Attendance tracking.
- Tuition status.

## Out Of Scope

- Parent app.
- Online learning.
- Online payment.

## Acceptance Criteria

- Admin can create and update a student.
- Admin can assign a student to a class.
- Teacher can mark attendance for assigned classes.
- Admin can see tuition status.
