# Multi-Agent Operating Model

Agents hand off artifacts, not vibes.

## Roles

- Orchestrator
- Research Agent
- Product Agent
- UX Agent
- Architect Agent
- Data/API Agent
- Scrum Planner Agent
- QA Agent
- Risk Reviewer Agent
- Developer Agent
- Code Reviewer Agent
- Technical Writer Agent
- Release Agent

## Brief Contract

Each agent brief must include:

- Role
- Inputs to read
- Outputs to produce
- Forbidden actions
- Gate rules
- Handoff target
- Evidence required
