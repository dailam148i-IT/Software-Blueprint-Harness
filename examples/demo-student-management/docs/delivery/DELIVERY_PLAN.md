# Delivery Plan

## Milestones
| Milestone | Goal | Exit Criteria |
| --- | --- | --- |
| M1 Discovery | clarify MVP and sensitive-data rules | PRD and MVP scope approved |
| M2 Blueprint | prepare implementation docs | lint, assess, and readiness run |
| M3 Student MVP | implement student profile workflow | US-001 evidence linked |

## Definition of Ready
- Requirement, acceptance criteria, story, API/spec links, test row, edge cases, and owner exist.

## Definition of Done
- Code, tests, docs, test matrix evidence, memory, and review findings are complete.

## Agile Rules
- Stories are vertical implementation slices.
- New scope requires a decision record.
- Only ready stories enter implementation.
